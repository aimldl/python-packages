# Pandas

Notes, code, and troubleshooting from learning Pandas, mainly by working through
[Pandas for Everyone: Python Data Analysis](https://www.amazon.com/Pandas-Everyone-Analysis-Addison-Wesley-Analytics-ebook/dp/B0789WKTKJ).

## Folder Structure
- [`book-pandas-for-everyone/`](book-pandas-for-everyone/) - notebooks, scripts, and data for the "Pandas for Everyone" book
- [`cheat-sheet/`](cheat-sheet/) - the official Pandas cheat sheet plus quick notes
- [`INSTALL.md`](INSTALL.md) - installation instructions
- [`how-to.md`](how-to.md) - short how-to snippets (e.g. reading files)
- [`troubleshoot.md`](troubleshoot.md) - notes on errors encountered and how they were resolved
- [`pd-read-csv-df-to-numpy.py`](pd-read-csv-df-to-numpy.py) - standalone example script
