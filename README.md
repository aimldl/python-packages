# python-packages
A sandbox environment for evaluating, testing, and learning third-party Python libraries.

## 🎯 Purpose
This repository isolates external dependencies from core logic. It serves as a scratchpad to test-drive new frameworks, understand their APIs, and build proof-of-concept scripts before integrating them into actual projects.

## 🗂️ Directory Structure
* **`/packages`**: Subdirectories organized by library name (e.g., `pandas`, `requests`, `pytest`), containing setup configurations and experimental scripts.
